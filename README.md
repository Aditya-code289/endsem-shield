
Just give some inputs and get minimum marks you need to not to get a back in endsem

EndSem Shield is a student-focused web tool that calculates the minimum marks required in the end-semester exam to avoid a back (fail), based on internal marks, lab marks (if applicable), and TA marks.

This project is a rebuilt and upgraded version of my very first JavaScript project — now redesigned with proper UI, dynamic rendering, and structured product thinking.
